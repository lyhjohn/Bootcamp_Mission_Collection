# 개요
간단한 커머스 프로젝트

목표: 셀러와 구매자 사이에서 중개해 주는 커머스 서버를 구축한다.

기술: Spring, Jpa, Mysql, Redis, Docker, AWS

## 회원

### 고객
- [ ] 회원 가입
